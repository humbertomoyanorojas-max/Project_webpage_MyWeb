# Humberto Moyano | Ebanistería Premium y Arquitectura Biodinámica

Este repositorio contiene el código fuente de la plataforma web premium del taller de ebanistería de autor y bioconstrucción **Moyano**. El sitio ha sido concebido como una vitrina digital de lujo diseñada con una estética inmersiva y un sistema interactivo enfocado en atraer inversiones patrimoniales y calificar clientes de manera inteligente.

---

## 🌟 Características Destacadas

*   **Experiencia Visual Inmersiva (Look & Feel de Lujo):** Diseñado sobre una paleta orgánica HSL en verde bosque profundo, acentuado con oro viejo y bronce. Cuenta con grano fotográfico sutil de fondo y efectos dinámicos de vidrio (*glassmorphism*).
*   **Portafolio de Obras Interactivo y Categorizado:**
    *   Filtros dinámicos al instante (**Mobiliario de Autor**, **Estructuras Biodinámicas**, **Fincas & Campo**).
    *   **Lightbox de Detalles Técnicos:** Un visualizador de gran formato que expone de forma transparente la ficha técnica de cada obra: especie de madera maciza (roble, nogal, teca, cerezo), acabados ecológicos libres de tóxicos (aceite de tung, cera de abejas), herramientas aplicadas y sellos de resistencia estructural generacional.
*   **Formulario Inteligente de Calificación Exclusiva (Filtro de Inversión):**
    *   Asistente multi-pasos interactivo que evalúa las prioridades del cliente.
    *   **Filtro Antidescarte de Bajo Costo:** Descalifica de forma elegante y automatizada a aquellos leads que buscan materiales comerciales industriales de corta duración (MDF, aglomerados, melaminas), educándolos sobre el compromiso de calidad del taller y preservando el tiempo de diseño para proyectos de verdadero valor patrimonial.
    *   **Rangos de Inversión Localizados:** Integración de montos reales y competitivos alineados con el mercado premium colombiano (COP).
*   **Responsividad Absoluta y Fluida:**
    *   Uso de tipografías fluidas con funciones CSS `clamp()` para asegurar legibilidad milimétrica en pantallas que van desde un iPhone SE hasta monitores ultra-panorámicos.
    *   Grids y espaciados totalmente flexibles que previenen desbordamientos.

---

## 🛠️ Tecnologías y Arquitectura

*   **HTML5:** Estructura semántica avanzada optimizada para motores de búsqueda (SEO).
*   **CSS3 (Vanilla):** Hojas de estilo autocontenidas basadas en variables nativas, transiciones fluidas de hardware, scrollbars de lujo y consultas de medios especializadas. **Sin frameworks adicionales ( Tailwind, Bootstrap, etc.)**, garantizando un rendimiento y control de píxeles máximo.
*   **JavaScript (ES6+):** Lógica nativa e interactiva sin dependencias externas pesadas, implementando Intersection Observers para efectos de revelado al hacer scroll y validación de formularios reactiva.

## 🚀 Despliegue en GitHub Pages (Paso a Paso)

Este sitio es una **Single Page Application (SPA)** completamente estática, lo que permite un despliegue gratuito, automático y de alta velocidad a través de **GitHub Pages**.

Para publicarlo al público de forma inmediata:
1.  Ingresa a la configuración del repositorio en GitHub (**Settings**).
2.  Desplázate en el menú lateral izquierdo hasta la sección **Pages**.
3.  Bajo la pestaña **Build and deployment**, selecciona la fuente **Deploy from a branch**.
4.  En la opción **Branch**, selecciona `main` (o `master`) y la carpeta `/ (root)`. Haz clic en **Save**.
5.  ¡Listo! En menos de un minuto, GitHub generará tu enlace público (ej. `https://humbertomoyanorojas-max.github.io/Project_webpage_MyWeb/`). Cada cambio que subas a este repositorio se reflejará en vivo de forma instantánea.
